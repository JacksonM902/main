What is my website siteplan?

What is a website's sitplan? Well, lets start with the storyboard. A website storyboard lets you see what the website will look like before
you build it. this helps guide you while you build the website. and helps keep your website looking organized and well-designed.
Next, the sitemap. A website sitemap lets you see what sites link to other sites, and what core features each site will have. for instance,
if you are creating a website that has 100 links in it, Don't you think that you might need something that keeps all the links organized?
thats where A website sitemap comes into play. It lets you see what links go to what websites, and just like a storyboard, it helps keep
you organized. Click here for some examples
